While I began my dev career mainly working on .NET/MSSQL desktop applications, over the last few years I've developed a strong interest in web dev. I've spent the last year learning all I can about Ruby on Rails, and attempting to get my HTML, CSS and Javascript up to a level I can be proud of.

If you want to know more about me and my work, feel free to check out my resume, github, Twitter and LinkedIn.
